Що таке GNU/Linux?

GNU/Linux – це операційна система на базі ядра Linux, яка поєднує в собі вільне програмне забезпечення GNU та інші компоненти. Вона широко використовується у серверах, комп'ютерах та інших пристроях.

Популярні дистрибутиви GNU/Linux

  • Ubuntu
  • Debian
  • Fedora
  • Arch Linux
  • Linux Mint
  • CentOS

Основні команди Linux CLI

  • pwd – показати поточний каталог
  • echo – вивести текст
  • ls – показати файли в поточному каталозі
  • cd – змінити каталог
  • touch – створити файл
  • mkdir – створити каталог
  • cp – копіювати файли або каталоги
  • mv – перемістити або перейменувати файли
  • rm – видалити файли або каталоги
  • history – показати історію команд

Простий BASH-скрипт для імітації авторизації

#!/bin/bash
echo "Введіть ім'я користувача:"
read username
echo "Введіть пароль:"
read -s password

if [[ $username == "admin" && $password == "1234" ]]; then
    echo "Авторизація пройшла успішно!"
else
    echo "Невірне ім'я користувача або пароль!"
fi
